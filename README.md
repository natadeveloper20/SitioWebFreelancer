# Freelancer Website 💻

Landing page para servicios freelance desarrollada con HTML, CSS y JavaScript.

Este proyecto fue creado para practicar el desarrollo de interfaces web modernas, responsive design y buenas prácticas de estructura HTML.

1. Clonar el repositorio
git clone https://github.com/natadeveloper20/SitioWebFreelancer.git
